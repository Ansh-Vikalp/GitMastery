Working with Branches

What are Branches?

Branches are used to develop features, fix bugs, or experiment with new ideas. They allow you to work on your code without affecting the main codebase. The default branch is usually named main or master.

Creating and Switching Branches

To create a new branch and switch to it, use the git checkout -b command:

git checkout -b

Listing Branches

To list all branches in your repository, use:

git branch

The currently active branch is indicated by an asterisk (*).
