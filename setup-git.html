Setting Up Git Locally

Installing Git

Git can be installed on various platforms. Here's how to set it up:

  • Download Git from the official site.Select the desired OS (Windows)
  • Now download the latest version 2.46.0 by selecting 64-bit GIT Windows setup if your system support 64 bit
  • Double click the installer in your downloads. Select the Drive (D:) where you want to install.
  • Run the installer and follow the prompts. Next -> Next -> Next -> Install

The git installer gives you three ways to use it

  1. GIT Bash Shell (Linux Users)
  2. GIT Cmd Shell (Windows Users)
  3. GIT GUI (GUI Users)

Verify the Git installation by opening up cmd prompt and executing: 

C:\Users\Asus\>git --version
git version 2.46.0.windows.1

Configuring Git

After installation, set up Git with your user information to track your commits.
These Configuration plays Important role in How your contribution are identified, tracked & merged within a repository and across collabrative platform like GitHub.

Different Levels of Setting:
  1. Local: Setting applies to only the specific repo where your cmd runs
  2. Global: Setting applies to all the repos on your loacl machine for one specific user that is logged-in.
  3. System: Setting applies to all the users on the syatem and to all their repos.
 
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Checking Configuration

Verify your Git configuration by running:

git config --list
