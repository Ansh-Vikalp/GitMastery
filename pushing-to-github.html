Pushing to GitHub & Collaborating

Pushing Commits

After committing changes locally, you can push them to a remote repository like GitHub:

git push origin

Pull Requests

When working with collaborators, you often make changes in a branch and push them to GitHub. You can then create a pull request to propose merging those changes into the main branch.

Pulling Changes

To fetch the latest changes from a remote repository and integrate them into your current branch, use:

git pull
