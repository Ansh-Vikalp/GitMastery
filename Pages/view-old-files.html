Get Old Version With Git Checkout

If our repository is quite large containing thousands of files and numeraous line of codes. Then using git show to view files in terminal and what changed has been made is not convinent. In such cases we use git checkout to get old version of a specific file or all files from that old commit in our working directory and those files will appear with a status of modified

To view all files in working dir of old commit:

git checkout <commit-Id> -- *

To view specific file in working dir of old commit:

git checkout <commit-Id> -- index.html

Now, to get back to our latest commit:

git checkout master -- *
git checkout master -- index.html
