Merging & Resolving Conflicts

Merging Branches

To combine changes from one branch into another, use the git merge command:

git merge

This will merge the specified branch into your current branch.

Fast-Forward Merge

If no new commits have been made to the current branch, Git will perform a fast-forward merge, simply moving the branch pointer forward.

3-Way Merge

If both branches have unique commits, Git will perform a 3-way merge. In cases where the same lines were modified in both branches, a merge conflict may occur.

Resolving Conflicts

If a merge conflict occurs, Git will prompt you to manually resolve it. After resolving conflicts, stage the file and commit the merge:

git add git commit
